:: Improvements​

History
  • In 1.6.0 and previous, History will can load only Design/UI of your app.
  • From 1.6.1 and later, it can load both Design and Features. Easily for you to design.

Save
  • Save button (recommended) will store all changes to cloud. Open your app on other laptops or FluxBuilder Web, you can see all the changes.
  • Apply button will apply your change only (on your local). Need to click Apply to apply your change, then Save.

Excluded Payment Method IDs

Some users want to hide some unnecessary payment methods (show webview) on the app through FluxBuilder, without changing the source code.

Version 1.6.0 - Desktop

Last updated: Nov 04th, 2022

✨View this FluxBuilder Blog for more detail.

(1) Quick Improve TikTok 🎨​

• Improve loading time for TikTok layout

• Please select TikTok Videos widget, change Username, input a Video ID and Update to active TikTok again:

(2) One Month Support plan 🎨​

After the free-support 6 months, if you need to extend your Technical Support period or change Domain, use the SUPPORT plan.

(3) Compatible with New Stuff 🎨​

  • Compatible with macOS Ventura 🎉

  • Compatible with the latest FluxStore v3.3.

(4) New AppBar template 🎨​

A new template supports 2-row appbar.

(5) New Shoes UI template ⚡️​

Available for Extended license and [Problem Internal Link].

(6) TabMenu & Scrollable 🚀​

  • Add new 2 UI complex layout types: TabMenu & Scrollable.

  • Use the Dynamic screens as the sub pages.

  • The Vertical layout is not used for these new UIs.

Video

image

(7) MultiLevel Category layout 🧩​

  • Category layouts has a new MultiLevel - useful to configure level of Categories.

  • Large number of Categories: useful for more than 100 Categories data.

Screenshot

(8) Vertical Product List ♻️​

  • New UI for Vertical Product List on Home page: Column With Filter.
Screenshot

(9) Phone Number Config ☎️​

On the Checkout screen, this feature helps to validate the phone number entered using Google's libphonenumber. Simpler and cheaper than sending OTP to verify phone number.

Features
  • Check validation of Phone Number.
  • Ability to customize Default Country.
  • Use for either normal or international phone numbers.
  • Support customizing the Country List.
  • Show or hide Flag Image.
  • Support RTL language.
  • Have 3 Selector Types for choosing country list.
Screenshot

(10) Hide/show Tabs ⭐️​

  • Easily show or hide Tabs from the bottom tab menu options.

  • Also can set the Default tab/screen.
