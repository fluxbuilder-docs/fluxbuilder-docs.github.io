## Version 2.0.9


Last updated: Aug 1th, 2024


[FluxBuilder.com](https://fluxbuilder.com/) | [MacOS](https://github.com/inspireui/fluxbuilder/releases/download/v2/fluxbuilder_macos.zip) |  [Window](https://github.com/inspireui/fluxbuilder/releases/download/v2/fluxbuilder_windows.zip)  | [Web](https://web.fluxbuilder.com/) | [Support](https://support.inspireui.com/) | [Video](https://www.youtube.com/playlist?list=PLcF-HiQy-jOK1yhvqE5aj_s4OXqrC9J1v) | [Affiliate](https://www.fluxbuilder.com/affiliate)


The latest version of FluxBuilder (2.0.9) introduces a major UI/UX redesign, new design template, resulting in a more intuitive and user-friendly app creation process.


### (1) Add more Templates for FluxNews ✨


We have added more templates for News/Blog apps, giving you more options to choose from to quickly design your app. The new templates come with sleek designs and enhanced functionality to help you create engaging content. 


![](/images/notion/615c853a4f5e.png)


### (2) Upgrade new Firebase Push Notification


We have upgraded the Firebase Push Notification feature in FluxBuilder, making it easier for you to send notifications to your app users. The new feature provides a more seamless integration with Firebase, allowing you to effortlessly send notifications


![](/images/notion/daf0084e2a07.png)


### (3) Automatic update Layout Items 


We have introduced an automatic update for design layout items, making it easier for you to keep your content up-to-date without manual intervention.


### (4) Improvements

1. Fix deleted Dynamic layout items

    We have addressed an issue where dynamic layout items could not be deleted. This fix ensures that your content is displayed correctly.

2. Upgrade Performance for Web Version ⚡️

    We have optimized the performance of the web version of FluxBuilder, resulting in faster loading times and a smoother user experience.
