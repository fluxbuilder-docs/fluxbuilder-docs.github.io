## Blog Detail Page


FluxBuilder support settings basically the Blog Detail Page. Open your app on FluxBuilder and get started as following.


On the simulator, click on any blog to open the Detailed Blog Settings. Now, enable any option you would like to display on the blog detail page, including:

1. Comment
2. Share Button
3. Text Zoom Button
4. Related Blog
5. Author and Post Date
6. Detailed Blog Layout: Simple, One Quater Image, Half Size Image and Full Size Image.

## Blog Integration


In case you would like to show blogs from other website, Blog Integration will handle that.

1. Open your app on FluxBuilder.
2. Go to Features and open Blog Integration.
3. Enable **Override Blog URL** option.
4. Input the website following the format `https://domainnname.com`
