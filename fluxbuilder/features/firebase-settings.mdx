Follow steps below to set up Firebase for your Apps:


## 1. Create Project on Firebase


Step 1: Open the Firebase page [https://console.firebase.google.com](https://console.firebase.google.com/) 


Step 2: Select the [Create a project] button 


![](/images/notion/162397efa9a7.png)


Step 3: Enter your Project name, tick all checkboxes, and click the Continue button:


![](/images/notion/b0d0b338a5fa.png)


Step 4: Click the Continue button. Then tick the [I accept…] checkbox, and select the [Create project] button:


![](/images/notion/86f2e00985e9.png)


Step 5: Click the Continue button:


![](/images/notion/8162643c9269.png)


That’s all. You have created your project on Firebase successfully.


## 2. Add an Android App into your Firebase project


**Step 1**: Open the Firebase page [https://console.firebase.google.com](https://console.firebase.google.com/) 


**Step 2**: Open your Firebase project. Click the gear icon in the upper left corner, and select the [Project settings] item:


![](/images/notion/506f2be68fb3.png)


**Step 3**: Select the Android icon in the [Your apps] section:


![](/images/notion/e54cc5f53bc7.png)


**or** select the [Add app] button, then click the Android icon:


![](/images/notion/901ec0f3ba77.png)


**Step 4**: Enter your [Android package name] and click the [Register app] button:
The [Android package name] and [Apple bundle ID] should be the same. 
_**Your Android package name must not include**_ [_**reserved keywords**_](https://dart.dev/guides/language/language-tour#keywords)_**. Use only lowercase letters.**_ 
Recommend in reverse domain format. For example, your domain is _example.com_, bundle ID and package name for your app should be "_com.example.app_"


![](/images/notion/f64ea1ae9e2a.png)


**Step 5**: Click the Next button. You can download the [google-services.json] file or do it later:


![](/images/notion/dbf9e12cde80.png)


**Step 6**: At the [Add Firebase SDK] section, click the Next button:


![](/images/notion/f4c14cd7fa3f.png)


**Step 7**: Click the [Continue to console] button. 
At this stage, it is possible that Firebase will not detect your test device, but that's okay, just click the [Skip step] button to continue the following steps. 


![](/images/notion/f6f13065741d.png)


You have added an Android app to your Firebase project successfully. Congratulations!


![](/images/notion/83f76215e6e3.png)


## 3. Add an iOS App into your Firebase project


**Step 1**: Open the Firebase page [https://console.firebase.google.com](https://console.firebase.google.com/) 


**Step 2**: Open your Firebase project. Click the gear icon in the upper left corner, and select the [Project settings] item:


![](/images/notion/506f2be68fb3.png)


**Step 3**: Select the iOS icon in the [Your apps] section:


![](/images/notion/dfd33ed294c2.png)


**or** select the [Add app] button, then click the iOS icon:


![](/images/notion/901ec0f3ba77.png)


**Step 4**: Enter your [Apple bundle ID] and click the [Register app] button:
The [Apple bundle ID] and [Android package name] should be the same. 
_**Your Apple bundle ID must not include**_ [_**reserved keywords**_](https://dart.dev/guides/language/language-tour#keywords)_**. Use only lowercase letters.**_ _(_[_This guide_](https://youtu.be/ZYQoKhsv2-I) _to creating iOS Bundle ID)._
Recommend in reverse domain format. For example, your domain is _example.com_, bundle ID and package name for your app should be "_com.example.app_"


![](/images/notion/ce50b87e0621.png)


App nickname is only used to distinguish between applications on firebase so you can set arbitrary.


**Step 5**: Click the Next button. You can download the [GoogleService-Info.plist] file or do it later:


![](/images/notion/723e5ae703f3.png)


**Step 6**: Click the Next button until you see the [Continue to console] button, click it as image below:
_(At this stage, it is possible that Firebase will not detect your test device, but that's okay, just click the [Skip the step] button to continue the following steps.)_


![](/images/notion/2d68eedf9404.png)


You have added an iOS app to your Firebase project successfully. Congratulations!


You can download the [GoogleService-Info.plist] file _(and save this important file to use later):_


![](/images/notion/116b2700f22a.png)


## 4. Enable Firebase on FluxBuilder


Let’s enable Firebase on FluxBuilder by steps:


Step 1: On FluxBuilder, open your app. 


Step 2: Select the **Features** on leftmenu. Click the **Firebase** button. 


Step 3: Turn on the **Enable Firebase** option.


![](/images/notion/9d8cc1809fb8.png)


That’s all :)
