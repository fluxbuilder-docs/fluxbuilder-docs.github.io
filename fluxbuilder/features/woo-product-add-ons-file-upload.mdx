This guide explains the way to configure WooCommerce Product Add-Ons, Upload File on the product (especially for drug store/ online medicine store, needs to upload prescription), short-description type, heading type; YITH WooCommerce Product Add-Ons.


## 1. Product Add-Ons Configuration


The Product Add-Ons plugin is supported on WooCommerce apps:


![](/images/notion/723630653755.gif)


To use this plugin, make sure you are using our latest MStore-API plugin which supports this feature. Older versions will not work. Refer to image below:


![](/images/notion/70208d4ca946.png)


Then you can choose to create either Global Product Add-Ons or Per-Product Add-Ons by following [their documentation](https://docs.woocommerce.com/document/product-add-ons/):


![](/images/notion/c732f6554d53.png)


**Supported types:** _Multiple Choice, Checkboxes, Short Text, Long Text, File Upload, Heading._


Other types are not supported.


![](/images/notion/d32bb1c70363.png)


**Please note that** the Product Add Ons that only compatible when the **[Enable OnePage Checkout] is Off**. 


Disable OnePage Checkout by following these steps:

1. Open your app on FluxBuilder.
2. Go to **Features**. In **Payment Methods**, open **Webview Payment**.
3. Turn off the option **Enable OnePage Checkout**.
4. Click **Save** button.

![](/images/notion/4b431f4d864c.png)


## 2. File Upload


Users can upload files to products (example: for drug store/ online medicine store, needs to upload prescription). This feature is available with WooCommerce ([single](https://1.envato.market/QL4d9) and [muli-Vendor](https://1.envato.market/NW4q1)) and compatible with [Product Add-Ons](https://docs.woocommerce.com/document/product-add-ons/).


![](/images/notion/24ff948e82b9.png)

1. Open your app on FluxBuilder.
2. Go to **Features**. In **General**, open **Product Add-Ons**.
3. Activate the types you need and adjust the **Upload Size Limit**.
4. Click **Save** button.

![](/images/notion/10864da9632d.png)


Once again, make sure the [Enable OnePage Checkout] is off:


![](/images/notion/795c569e8644.png)


## 3. Yith WooCommerce Product Add-Ons


FluxStore from version 5.4.0 supports the YITH WooCommerce Product Add-Ons plugin on Woocommerce site, allowing merchants to offer customizable product options such as checkboxes, radio buttons, text fields, and more directly within the app.


This integration enhances the product configuration experience and enables more flexible product offerings — ideal for stores that sell personalized or build-to-order items.


Install the plugin [YITH WooCommerce Product Add-Ons](https://en-gb.wordpress.org/plugins/yith-woocommerce-product-add-ons/) and then start using the feature on the app.
