The payment is available for Woocommerce, Pro (Woocommerce, Opencart), and Multi Vendors apps.


## Configure Thawani Payment


Before configuration on app, you can choose to install the plugin [Thawani for Woocommerce](https://github.com/PhazeRoOman/thawani-for-woocommerce) on your website (this is optional).


If you are using OpenCart, you must install the [Thawani Oman Payment Gateway](https://www.opencart.com/index.php?route=marketplace%2Fextension%2Finfo&extension_id=41825&filter_search=thawani) extension on your website.



Then use [FluxBuilder](https://www.fluxbuilder.com/download):

1. Open your app on **FluxBuilder**
2. Go to **Features** → **Payment Methods** → **Thawani Payment**
3. Turn on **Enabled Thawani Payment** and input **Secret Key**
4. If using **Promotion Mode**, please enable **Promotion Mode** and input **Publishable Key**, otherwise please disable **Promotion Mode.**

 _Refer to_ [_this link_](https://thawani-technologies.stoplight.io/docs/thawani-ecommerce-api/7c0f75e1668d7-thawani-test-card) _for Thawani Test Card._
