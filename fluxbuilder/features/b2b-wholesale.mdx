title: "B2B WholeSale"
---


The feature now supports Simple and Variable products.


The customers, after signing up as Wholesale Customers, can see the wholesale prices of the setup products.


## Prerequisite


The feature is available for WooCommerce, Pro(WooCommerce) and Multi Vendors apps.


Before setting up the feature, you need to install the [WooCommerce Wholesale Prices](https://wordpress.org/plugins/woocommerce-wholesale-prices/) plugin on your website to support the feature on the Fluxstore App.


Then follow the guide below:


## Configuration


### 1. Set Up the Product on Website


On your website, you will go to **Products** on the left side menu.


![](/images/notion/7fa13bc3668e.png)


Select the products that need setting Wholesale Price. Update the **Wholesale Price** for the **Wholesale Customer**.


![](/images/notion/63769e7f24a5.png)


### 2. Set Up the Feature on App

1. Open your app on FluxBuilder.
2. Go to **Features**, in **Cart & Checkout**, open **B2B Features**.
3. Turn on **Enable B2B** option.
4. Click **Save** button.

![](/images/notion/ea1a11585464.png)
