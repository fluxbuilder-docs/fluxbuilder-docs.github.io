FluxBuilder is offered in both desktop and web versions, giving you flexibility in how you choose to use it.


## 🖥️ Desktop version


### 1. Download

- Link to download: [https://www.fluxbuilder.com/download](https://www.fluxbuilder.com/download)
- To use FluxBuilder, ensure that your system is **macOS (64-bit)**, **Windows 10 or 11 (64-bit)**, or **x86-64 based**.

### 2. Installing on MacOS


**Requirement**: Your system is macOS (64-bit).

**Manual guide**:


![](/images/notion/2393ad46f853.png)


To install **FluxBuilder**, please follow these steps:

1. Double-click on `FluxBuilder_macOS.zip` to unzip its contents.
2. Right-click on `FluxBuilder.pkg` and select "Open" to install it. You may need to go to Apple > System Settings > Privacy & Security > Go to the end of the screen and allow installing FluxBuilder from here. Refer to this screenshot [https://snipboard.io/HF9eCk.jpg](https://snipboard.io/HF9eCk.jpg)
3. Click "Continue" until the "The installation was successful" message displays.
- Select `Close` and `Move to Trash` (or `Keep`) to delete (or keep) the FluxBuilder.pkg

    ![](/images/notion/21df3c4ae3f7.png)

- Press the `Command` and `Space` keys simultaneously to open Spotlight Search, then type "FluxBuilder" to run the FluxBuilder tool. Alternatively, you can find it in the Applications folder.

    ![](/images/notion/9fe0e221c0c9.png)


---


### 3. Installing on Windows


If your system is **Windows 10 or 11 (64-bit), x86-64 based**, follow these steps:

1. Unzip the `Fluxbuilder_windows.xxx` file.
2. Open the `Release` folder, double-click the `Fluxbuilder_Setup` file, and then select `More Info`. Click `Run anyway` and then click `Next` until FluxBuilder has been successfully installed.

---


### 4. Upgrade latest FluxBuilder


If you are using an older version, follow the steps below to upgrade:

1. [Export](https://docs.fluxbuilder.com/import-export-and-other) your customized design and features by saving the env.dart and config JSON file.
2. Delete the current FluxBuilder application from your PC or laptop.
3. Install the latest version of FluxBuilder.

## 🛠️ Common installation issues

- Failed to install on Mac: Right-click on FluxBuilder.pkg > **select Open** to install it. You may need to go to Apple > System Settings > Privacy & Security > Go to the end of the screen and allow installing FluxBuilder from here. Refer to this screenshot [https://snipboard.io/HF9eCk.jpg](https://snipboard.io/HF9eCk.jpg)
- FluxBuilder may start and then immediately close on Windows. To resolve this issue, ensure that your system is running Windows 10 or 11 (64-bit) on an x86-64 based processor. Then, try one of the following solutions:
    - **Solution 1**
        1. Open Run (by pressing `Window + R`)
        2. Enter this: `%appdata%`
        3. Delete the folder `com.inspireui.fluxbuilder` inside there to clear FluxBuilder settings.
    - **Solution 2**

        Please install the [Webview library](https://developer.microsoft.com/en-us/microsoft-edge/webview2/#download-section) shown in [this image](https://tppr.me/JhY6p). If you are still unable to open FluxBuilder, please follow these steps:

        1. Open Windows Update Settings > Click [Troubleshoot](https://imgur.com/dNA96b2) on the left menu.
        2. Tap to [Additional Troubleshooters](https://imgur.com/i4fshPU) > Tap to [Program Compatibility Troubleshooter](https://imgur.com/yCF8K7l) > Click Run the troubleshooter.
        3. In new window, choose [FluxBuilder](https://imgur.com/mzAoqQ3) > Next > Try recommended settings > [Check and test program](https://imgur.com/zTczMXw) > Next > Yes > Close. Now you have resolved conflicts between FluxBuilder and other applications on your Windows.
- FluxBuilder shows error about _`VCRUNTIME140_1.dll`_ _file, check below solution:_

    Go to `www.google.com`, search `VCRUNTIME140_1.dll` file, download that dll file and copy to `C:\Window\system32`, Restart Windows PC.

- FluxBuilder shows error about MSVCP140.dll, check below solution:

    To add the necessary file, download it from [https://www.dll-files.com/msvcp140.dll.html](https://www.dll-files.com/msvcp140.dll.html). Once downloaded, add it to the FluxBuilder application folder (usually located in _Program Files (x86)/InspireUI/FluxBuilder/fluxbuilder_).
