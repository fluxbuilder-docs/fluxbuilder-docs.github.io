<Info>
You can view a demo app of your website for Free using the drag&drop FluxBuilder App Builder. 
</Info>


<Warning>
Non-work features are NORMALLY because they require additional configuration (e.g. Map, Social Login, Push Notifications, Dynamic Link, Realtime chat, etc.).
So if they don’t work on your App from Test APK or Test iOS, it is PROPERLY because they need to configure more.
</Warning>
