> [Legacy custom apps can’t be created after January 1, 2026](https://changelog.shopify.com/posts/legacy-custom-apps-can-t-be-created-after-january-1-2026)

**Update:**

Starting January 1, 2026, you can no longer create new [custom apps](https://help.shopify.com/en/manual/apps/app-types/custom-apps) in the Shopify admin. This affects merchants and developers who create custom apps directly from the admin. Existing custom apps aren’t affected and will continue to work. To create a new custom app after this date, use the [Dev Dashboard](https://shopify.dev/docs/apps/build/dev-dashboard) to build the app, then install it on your store. This change improves security and long‑term support for custom apps.
</Warning>

### 2. Manual Guide 


To create the Storefront Access Token for your app, please follow these steps:

1. If you use **Shopify Partners**, go to **Stores** and select the store you want to set up. If you are logging directly into your **Store Dashboard**, you can skip this step.

    ![](/images/notion/79c8cb388e3f.png)

2. Open your dashboard store, go to **Apps** →  search `Headless` → Click to open [the app](https://apps.shopify.com/headless)

    ![](/images/notion/85dc2146788b.png)

3. Click on `Install` button

    ![](/images/notion/525f75b11a78.png)

4. After the Storefronts section opens, click on `Create storefront` button

    ![](/images/notion/8c0fab86dc86.png)

5. Click on `Manage` button to access `Storefront API`

    ![](/images/notion/8726d43a0315.png)

6. Click on the `Edit` button to update the Storefront API permissions

    ![](/images/notion/7083b57b6651.png)

7. Select all for the access scopes, next click **Save**

    ![](/images/notion/4ec25c10917a.png)

8. Finally, click on the copy button to copy the access token and paste into your Shopify app

    ![](/images/notion/4b973eca01cb.png)

9. Here is the result on FluxBuilder

    ![](/images/notion/222c078fb5d2.png)
