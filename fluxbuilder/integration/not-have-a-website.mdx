---
title: "Not Have A Website"
---


If you do not currently have a website but wish to build your own **iOS and Android applications**, please consider one of the following solutions:

- **Option 1**: For users who need to create standalone **iOS and Android mobile applications** without a linked website, we highly recommend our [**Serverless App**](https://docs.fluxbuilder.com/setup-serverless-store-in-admin-panel/) solution.
- **Option 2**: Subscribe to the [Shopify](https://shopify.com/) service to create your Shopify store, then use FluxBuilder to build both iOS and Android apps.
- **Option 3**: Set up new hosting and install a website, then use FluxBuilder to build both iOS and Android apps. We recommend you to install WooCommerce website.
