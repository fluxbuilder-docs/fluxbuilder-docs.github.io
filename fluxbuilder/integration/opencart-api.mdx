To build your Opencart app, install the API below on your Opencart website so that the app can sync with your data:


**Open the admin panel, then click** **`Extensions->Installer`** **and upload this** [**Opencart Extension file**](https://drive.google.com/file/d/14_5NmGHI0yLNhtFX-A8i1zIkxfAQeSK2)**.**
This plugin is required for supporting RestAPI to connect the app, support Payment, SMS Login, Facebook Login, and other features.
