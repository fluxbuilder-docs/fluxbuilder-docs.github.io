## 1. Shipping Methods

> **S**et up delivery options, and fees

### Add shipping method

- Go to **Shipping Methods** page.
- Input **Shipping Method Label**, **Shipping Cost**, and **Shipping Tax**.
- Click **Add Shipping Method** button.

![](/images/notion/d532f4f21a10.png)


### Edit shipping method

- On **Shipping Methods** page, click Edit icon button of an existing shipping option.

![](/images/notion/9642fff54791.png)

- Change the old settings and click **Update Shipping Method** button to apply.
- Click on the Delete icon button of a shipping option to remove it.

![](/images/notion/4172bf839753.png)


## 2. Payment Methods

> Set up payment method id, and more information about the payment

### Add payment method

- Go to **Payment Methods** page.
- Input **Payment Method ID**, **Payment Method Title.**
- Upload image and add **Payment Method Description** (if any)
- Lastly, click **Add Payment Method** button.

![](/images/notion/308f84acf042.png)


### Edit payment method

- On the **Payment Methods** page, click Edit icon button of an existing payment option.

![](/images/notion/e146ee15e423.png)

- Change the old settings and click **Update Payment Method** button to apply.
- Click on the Delete icon button of a payment option to remove it.

![](/images/notion/e8dd3553f96b.png)
