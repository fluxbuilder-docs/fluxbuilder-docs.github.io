Create Serverless App

Follow this guide to quickly set up a new serverless app using FluxBuilder.

Precondition​

Before you begin, make sure to:

  1. Install the latest version of FluxBuilder on your device.
  2. Sign Up/ Sign In to your account on FluxBuilder.

Step-by-Step Instructions​

  • Click Create New App on the Dashboard page.

  • Select the No Hosting option.
  • Sign in to your Firebase Google account.

  • Input your Firebase Project ID (or click Random button to get the random ID)

  • Click the VALIDATE PROJECT ID button (it may take time at this step to complete the validation process).
  • After the successful validation, the Initiating Firebase Data will display (follow the provided instructions to complete the setup on Firebase Console).
  • Then, click SETUP PROJECT button.

  • After finishing the setup process, click Next to continue.
  • In the next step, select Templates for your app. Continue to click Next.

  • In the next page, input the App Name and other fields (optional). Click Next.

  • Then, click Create App button.

  • Once the app is successfully created, navigate to the Admin section to configure your store within the app.
