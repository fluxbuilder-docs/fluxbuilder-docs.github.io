Download and Install

FluxBuilder is offered in both desktop and web versions, giving you flexibility in how you choose to use it.

1. Downloadâ€‹

  • Link to download: https://github.com/inspireui/fluxbuilder/releases
  • To use FluxBuilder from InspireUI, ensure that your system is macOS (64-bit), Windows 10 or 11 (64-bit), or x86-64 based.
  • The simulator on FluxBuilder only displays the app design, not its features. Therefore, please refrain from clicking to test features on the simulator. Instead, to test the features, please try the app demo.

2. Installing on MacOSâ€‹

Your system is macOS (64-bit).

To install FluxBuilder, please follow these steps:

  1. Double-click on FluxBuilder_macOS.zip to unzip its contents.
  2. Right-click on FluxBuilder.pkg and select "Open" and then "Open" again.
  3. Click "Continue" until the "The installation was successful" message displays.

3. Installing on Windowsâ€‹

If your system is Windows 10 or 11 (64-bit), x86-64 based, follow these steps:

  1. Unzip the Fluxbuilder_windows.xxx file.
  2. Open the Release folder, double-click the Fluxbuilder_Setup file, and then select More Info. Click Run anyway and then click Next until FluxBuilder has been successfully installed.

4. Upgrade latest FluxBuilderâ€‹

If you are using an older version, follow the steps below to upgrade:

  1. Export your customized design and features by saving the env.dart and config JSON file.
  2. Delete the current FluxBuilder application from your PC or laptop.
  3. Install the latest version of FluxBuilder.

  • Failed to install on Mac: Right-click on FluxBuilder.pkg > select Open > Open again. It will work.

  • FluxBuilder may start and then immediately close on Windows. To resolve this issue, ensure that your system is running Windows 10 or 11 (64-bit) on an x86-64 based processor. Then, try one of the following solutions:

    • Solution 1

      1. Open Run (by pressing Window + R)
      2. Enter this: %appdata%
      3. Delete the folder com.inspireui.fluxbuilder inside there to clear FluxBuilder settings.

    • Solution 2

      Please install the Webview library shown in this image. If you are still unable to open FluxBuilder, please follow these steps:

      1. Open Windows Update Settings > Click on the left menu.
      2. Tap to Additional Troubleshooters > Tap to Program Compatibility Troubleshooter > Click Run the troubleshooter.
      3. In new window, choose FluxBuilder > Next > Try recommended settings > Check and test program > Next > Yes > Close. Now you have resolved conflicts between FluxBuilder and other applications on your Windows.
