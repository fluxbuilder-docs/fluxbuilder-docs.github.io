Settings

:::💫

In this section, you can configure all information for Your App Builder, such as: name, color, logo, show or hide blog, products on Dashboard, etc.

⛳️ After completing the Settings, scroll to the end of screen to click Save.

:::

Agency​

1. Name & Description​

In this part, you will need to input the Agency Name to show on the AppBuilder.

The Background Image is set to show on Login screen.

2. Logo & Color​

  • The Main Color will change all the color of all features available on your AppBuilder. Let select the color that show off your style and attract the users.
  • The Logo is located beside the Agency Name. You can upload the 2 separate logos for Light Theme and Dark Theme
  • Preview Logo is your wonderful logo which will display on the simulator as a Preview Layout while user configure the app design (before they upload their own logo).

:::📌

Remember to upload your Preview Logo, otherwise, our FluxStore logo will display instead.

:::

3. Product Settings​

This Settings is for Products (on the left menu)

  • In case you would like to present your existing stunning design apps on Customer Dashboard Screen, turn on Enable. If not, please disable it by turn off the option.
  • Do not forget to input Label or you can ignore it if needed.
  • Wrap Layout will help to display full of your Products on one page. If not, some products will be hidden and users need to scroll horizontally on the list to see the rest.

4. Advance Settings​

  • Manage the submitting Build action of user by Allow Customer to Submit Build option
  • Send notification to users (displaying on Dashboard screen) by adding your Wordpress website URL to Notification.

5. Wordpress Extension​

As your experience with FluxStore apps, you will know Mstore API plugin which supports most of the app Features. To be more convenience when supporting your users, the Agency AppBuilder will export the Mtore API plugin under your Wordpress Extension.

Input your Plugin Name → click Save and click Generate Plugin to get it.

6. Blog​

Add the Blogs/News of your Wordpress website on the Dashboard Screen of User

  1. Click Add +
  2. Input Name (label) and your Wordpress URL
  3. Select Layout:
  • simple

  • titleLeft

  1. Enable Skeleton to show skeleton while loading blogs on screen

  2. Click ADD

:::📌

You can configure each existing blog by clicking the Edit button and remove it by clicking the Delete button

:::

Billing​

Instead of opening FluxBuilder to Cancel or manage your Subscription, you can open Billing to do that.

Cancel your Subscription​

Click Cancel on Billing screen to discard the next payment. You will have access to your subscription until the end of the current billing cycle.

Change the Subscription plan​

To change to another Subscription plan, please make sure that you have canceled the existing subscription package.

After subscribe successfully, you need to reload the Admin page to see the change.
